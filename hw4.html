This is our implementation of an IK solver using a gradient descent algorithm and Jacobian matrices.
We followed the method using the pseudo-inverse of the Jacobian as detailed on piazza.
The root link is attached to an immobile ball joint, the solver works with out-of-reach goals, and it also works
on goals that take it through the base joint. Aside from working with any number of joints and any length arms,
it also is able to trace out any goal path.
